# Support Replay

Turn recurring support pain into a fixable product backlog.

Generated by Android Autopilot.

## Target user
Founders and small support/product teams handling user issues directly.

## Problem
Repeated support complaints live in inboxes and chats, making it hard to identify patterns and prioritize product fixes.

## Solution
A mobile-friendly issue pattern board that groups recurring user complaints into themes, evidence, and proposed fixes.

## V1 features
- Issue themes with frequency and severity
- Evidence snippets and notes
- Suggested fix backlog
- Weekly trend summary cards
- Offline sample dataset

## Monetization
Solo free tier, paid workspace features later.

## Package name
`com.harshitlabs.supportreplay`

## Run later

```bash
npm run start
```
