# Deprecation Radar

Track API, SDK, and vendor change risk before it breaks your product.

Compact Expo React Native MVP generated by Android Autopilot.

## What is included
- Command Center dashboard with 7d / 30d / 90d risk filters
- Vendor watchlist cards with owners, criticality, and migration progress
- Timeline + notes section with offline-first JSON handoff guidance
- Sample privacy notes and TODO markers for later storage/notification work

## Files
- `App.tsx` main Android MVP screen
- `src/theme.ts` design tokens
- `src/data/features.ts` sample vendors, inbox items, features, screens, and privacy notes

## Run later
```bash
npm install
npm run start
```
