# ShipPing

Monitor your product's critical endpoints and cron jobs from your phone.

Generated by Android Autopilot.

## Target user
Indie hackers, solo founders, and small developer teams running APIs, web apps, or scheduled jobs.

## Problem
Small teams discover downtime too late because many monitoring tools are desktop-first, noisy, or too expensive for tiny products.

## Solution
A mobile-first monitoring companion that shows endpoint health, cron freshness, SSL expiry, and incident notes in a lightweight dashboard.

## V1 features
- Project-based endpoint tracking
- Manual HTTP health checks and latency snapshots
- Cron heartbeat freshness tracker
- SSL expiry reminder section
- Incident notes timeline
- Offline cache of latest known status
- JSON export/import for configuration

## Monetization
Free tier for a few projects, then subscription or one-time Pro unlock for unlimited projects and cloud backup.
