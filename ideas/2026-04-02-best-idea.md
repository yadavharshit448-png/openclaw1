# Deprecation Radar

**Tagline:** Track API, SDK, and vendor change risk before it breaks your product.

## Target user
Solo founders, indie hackers, and small developer teams that rely on third-party APIs, SDKs, and SaaS tools.

## Problem
Small teams miss deprecation notices, pricing changes, SSL expiries, and vendor status updates because those updates are scattered across docs, changelogs, emails, and status pages.

## Solution
Deprecation Radar is a mobile command center where users track critical vendors, log version deadlines, watch changelog or status URLs, and receive simple reminders before a dependency change turns into an outage or fire drill.

## V1 features
- Add vendors with product name, category, and criticality
- Save changelog, docs, or status page links per vendor
- Manual deadline tracker for API version sunsets, renewal dates, and SSL expiry
- Notes timeline for pricing changes, migration steps, and rollout blockers
- Priority inbox of upcoming risks in the next 7/30/90 days
- Push reminders for due dates and saved follow-ups
- Offline local cache with JSON export/import
